git config

este comando nos sirve para configurar ya sea el usuario todo para configurar en la aplicacion git el cual es muy importante la configuracion de todo aspecto hecho a nuevo

git init

el git init es un coamando para inicializar un archivo o se podria decir para ubicar el documento con el cual se quiere trabajar el cual sirve para inicializar
El comando git init crea un nuevo repositorio de Git. Puede utilizarse para convertir un proyecto existente y sin versión en un repositorio de Git, o para inicializar un nuevo repositorio vacío.

git add

el git add nos sirve para guardar el documento o como tambien llamar al documento y subirlo al git hub pero hay que tener en cuenta que esto es una serie de pasos el cual este comando solo esta llamndo al documento y aun no se a guardado en este comando se anota git add y luego el nombre del archivo el cual este comando lo leera.

git status

el git status nos sirve para ver el estado del documento el cual al poner el comando nos da la señal si el archivo esta modificado o como tambien ya esta guardado lo modificado el cual una vez hecho esto se puede pasar al siguiente nivel

git push origin master
